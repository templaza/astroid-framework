[![Watch Intro Video here](https://www.joomdev.com/images/githubassets/astroid-play.jpg)](https://www.youtube.com/watch?v=ZflNq3fDaXE)


## Requirements
* Joomla: 3.x, 4.x
* PHP : 5.6+
